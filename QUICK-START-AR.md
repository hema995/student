# دليل البدء السريع - نظام إدارة بيانات الطلاب

## تشغيل التطبيق

### الطريقة 1: التطوير (مع أدوات المطور)

```bash
npm run dev:electron
```

هذا سيفتح نافذة Electron مع أدوات المطور مفعلة.

### الطريقة 2: الإنتاج (بعد البناء)

```bash
npm run start:electron
```

## استخدام التطبيق

### 1. إضافة طالب جديد
- انقر على زر "إضافة" في الأعلى
- املأ البيانات المطلوبة
- انقر "إضافة"

### 2. استيراد البيانات من Excel
- انقر على زر "استيراد"
- اختر ملف Excel/CSV/JSON
- أدخل اسم المجموعة
- انقر "استيراد الآن"

### 3. البحث عن طالب
- اختر نوع البحث (رقم قومي أو اسم)
- اكتب في مربع البحث
- انقر على الطالب من القائمة

### 4. تعديل بيانات طالب
- اختر الطالب من القائمة
- انقر زر "تعديل"
- عدّل البيانات
- انقر "حفظ التغييرات"

### 5. طباعة نماذج
- اختر الطالب
- انقر على الزر المناسب:
  - "طلب تحويل" - لطلب التحويل
  - "إنذار" - للإنذار بالفصل
  - "بيان قيد" - لبيان قيد الطالب
- املأ البيانات الإضافية
- انقر "طباعة"

### 6. حذف طالب
- اختر الطالب من القائمة
- انقر زر "حذف الطالب" في الأسفل
- أكد الحذف

## ملاحظات هامة

- ✅ جميع البيانات محفوظة تلقائياً
- ✅ لا يحتاج اتصال بالإنترنت
- ✅ البيانات محفوظة عند إغلاق التطبيق
- ✅ يمكنك استيراد بيانات غير محدودة

## موقع ملف البيانات

### Windows
```
C:\Users\[username]\AppData\Roaming\student-management-system\students.db
```

### Linux
```
~/.config/student-management-system/students.db
```

### Mac
```
~/Library/Application Support/student-management-system/students.db
```

## النسخ الاحتياطي

### إنشاء نسخة احتياطية
1. اذهب إلى مجلد البيانات أعلاه
2. انسخ ملف `students.db`
3. احفظه في مكان آمن

### استعادة من نسخة احتياطية
1. اذهب إلى مجلد البيانات
2. استبدل ملف `students.db` بالنسخة الاحتياطية
3. أعد تشغيل التطبيق

## المشاكل الشائعة

### المشكلة: التطبيق لا يفتح
**الحل**: تأكد من تثبيت Node.js 20 أو أحدث

### المشكلة: البيانات اختفت
**الحل**:
1. تحقق من وجود ملف `students.db`
2. استعد من النسخة الاحتياطية

### المشكلة: خطأ في الاستيراد
**الحل**:
1. تأكد من أن الملف بصيغة Excel/CSV/JSON
2. تأكد من وجود عمود "الاسم" و "الرقم القومي"
3. تأكد من أن الرقم القومي فريد (غير مكرر)
