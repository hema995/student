# نظام إدارة بيانات الطلاب - تطبيق Electron

## المميزات الجديدة

التطبيق الآن يعمل كتطبيق سطح مكتب مستقل باستخدام Electron مع:

- **عمل أوفلاين كامل**: لا يحتاج إلى اتصال بالإنترنت
- **قاعدة بيانات محلية**: تخزين البيانات باستخدام SQLite
- **حفظ تلقائي**: جميع البيانات محفوظة محلياً على جهازك
- **استمرارية البيانات**: عند إغلاق وفتح التطبيق، جميع بياناتك محفوظة

## كيفية التشغيل

### 1. التطوير (Development)

```bash
npm run dev:electron
```

هذا الأمر سيبدأ:
- خادم التطوير (Vite) على المنفذ 5173
- تطبيق Electron بنافذة المطور

### 2. البناء (Build)

```bash
npm run build:electron
```

هذا الأمر سيبني:
- التطبيق الكامل في مجلد `dist/public`
- ملف تنصيب للتطبيق في مجلد `release`
  - **Windows**: ملف `.exe` للتنصيب
  - **Linux**: ملف `.AppImage`
  - **Mac**: ملف `.dmg`

### 3. تشغيل التطبيق المبني

```bash
npm run start:electron
```

## موقع تخزين البيانات

البيانات محفوظة في:

### Windows
```
C:\Users\[اسم المستخدم]\AppData\Roaming\student-management-system\students.db
```

### Linux
```
~/.config/student-management-system/students.db
```

### Mac
```
~/Library/Application Support/student-management-system/students.db
```

## ملاحظات مهمة

1. **جميع البيانات محفوظة محلياً** على جهازك في ملف `students.db`
2. **لا حاجة للإنترنت** بعد بناء التطبيق
3. **النسخ الاحتياطي**: يمكنك نسخ ملف `students.db` لإنشاء نسخة احتياطية
4. **استعادة البيانات**: استبدل ملف `students.db` بالنسخة الاحتياطية لاستعادة البيانات

## الفرق بين النسخة السابقة والجديدة

| الميزة | النسخة القديمة (Web) | النسخة الجديدة (Electron) |
|--------|---------------------|---------------------------|
| الاتصال بالإنترنت | مطلوب | غير مطلوب |
| قاعدة البيانات | PostgreSQL (سحابية) | SQLite (محلية) |
| حفظ البيانات | على الخادم | على جهازك |
| التشغيل | متصفح الإنترنت | تطبيق سطح المكتب |
| الأمان | يحتاج حساب | بيانات محلية فقط |

## المشاكل الشائعة

### التطبيق لا يبدأ
تأكد من تشغيل:
```bash
npm install
```

### البيانات لا تظهر بعد إعادة التشغيل
تحقق من أن الملف `students.db` موجود في مجلد البيانات

### خطأ في البناء
تأكد من تشغيل:
```bash
npm run build
```
قبل تشغيل `npm run build:electron`
